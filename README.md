# Learning

AI Engineering learning hub covering the four disciplines (Prompt, Context, Workflow, and Specification Engineering), five autonomy levels, and structured hands-on sessions.

## Structure

```
Learning/                 Core learning materials and session content
  ai-engineering-disciplines.md   Entry point — disciplines, levels, role guides
  sessions.md                     Session index
  session-one/                    Session 1: AI Engineering Discipline + Statusline setup
References/               Supporting frameworks and research
  AIEngineering/          Competency framework, progression levels, self-assessment
  AIResearch/             Greenfield vs brownfield, intent-driven adoption
  SoftwareDevelopment/    Developer competency framework, progression, self-assessment
Slide-Deck/               Presentation PDFs
scripts/                  Utility scripts (PowerShell)
```

## Getting Started

Start with [AI Engineering Disciplines](Learning/ai-engineering-disciplines.md) for the full overview, or jump to [Sessions](Learning/sessions.md) for structured learning with hands-on exercises.
