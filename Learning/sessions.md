# Learning Sessions

## Overview

This document is the index for all learning sessions. Each session is a self-contained unit with three parts: a **Showcase** (concepts), a **Hands-On** (practical exercise), and **FAQs** (common questions). Sessions are numbered sequentially and live under `Learning/session-{number}/`.

## Context

The AI Engineering material in this hub covers a broad discipline with multiple roles, levels, and workflows. Sessions provide a structured learning path through that material -- each one pairs a conceptual introduction with a hands-on exercise so the ideas are grounded in practice rather than theory alone. See [ai-engineering-disciplines](ai-engineering-disciplines.md) for the foundational material the sessions build on.

> [!important]
> When a new session is added, update the session table below with its number, title, and status.

Every session folder follows the same layout:

```

  faqs.md           -- Common questions from the session
```
